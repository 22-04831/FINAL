Publishing Notes

Upload this entire folder to a GitHub repository. In repository settings, enable GitHub Pages and set the source to the main branch root folder. The site will use index.html as the landing page.

For large report PDFs, upload them to Google Drive and link them inside this archive page if needed. This keeps the GitHub website fast and lightweight.
