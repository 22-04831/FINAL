# Xyrra De Silva Learning Portfolio Website

This is a multi-page HTML/CSS/JavaScript website for the EE 417 Learning Portfolio.

## Pages
- `index.html` - Home dashboard
- `about.html` - Student profile
- `features.html` - Website features
- `seminars.html` - Seminar learnings
- `colloquia.html` - Colloquia summaries
- `matrix.html` - Learning matrix
- `reflections.html` - Personal reflection
- `evidence.html` - Evidence gallery using images rendered from uploaded reports
- `archive.html` - Reports archive and deployment notes

## How to publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files and folders from this folder to the repository root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save and wait for GitHub Pages to publish the site.

## Notes
- The evidence images are stored in `assets/evidence`.
- The original PDF reports are not included to keep the GitHub site lightweight.
- You may upload the PDF reports to Google Drive and link them from `archive.html`.
